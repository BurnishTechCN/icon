Burnish Icon

图标列表

使用

webpack方式:

  • npm install burnish-icon --save
  • import 'burnish-icon/style.css';
  • <i class="icon-1" />

普通方式:

  • 下载新版的icon包
  • 引入style.css和font文件夹
  • <i class="icon-1"></i>

制作icon

  • 找设计师做好svg。
  • 下载selection.json.bak文件。
  • 删掉.bak后缀，到IcoMoon，点击【import icons按钮】导入selection.json（先删掉所有已有的图标）。
  • 可以再次点击import icons按钮】导入自己新增的svg文件。
  • 确保导入的图标都处理点亮状态后，点击【generate font】按钮，导出字体图标压缩包。
  • Fork icon代码仓库到本地。
  • 替换项目中的font文件夹和style.css，测试确认无误后push到origin，并提pull request。
