使用（WEB端）

  • 下载新版的 icon 包，或者通过 npm install burnish-icon 安装。 得到里面的 s.svg 和 style.css
  • 引入 style.css 和 s.svg 文件
  • 加入代码： 
    <svg class="icon icon-1"><use xlink:href="s.svg#icon-1"></use></svg>
  • 自定义样式： 
    .icon-1 {
  fill: #656565;
  width: 16px;
  height: 16px;
}

制作icon

  • 找设计师做好svg。
  • Fork icon 代码仓库到本地。
  • 在svg目录导入新增的svg文件。
  • 到IcoMoon，先删掉已经存在的 icon。然后点击【import icons按钮】导入所有的 svg 文件。
  • 确保导入的图标都处理点亮状态后，点击【Generate SVG】按钮，点击【download】导出压缩包。
  • 替换项目中的 s.svg
  • 打开 s.svg ,去掉里面的 fill="#xxx" 属性。
  • 修改 doc/data.js 配置，加入新增的 icon class。
  • 执行 python -m SimpleHTTPServer 9999，访问 http:localhost:9999 看效果。
  • 测试无误后，push 代码，提交 pull request。
