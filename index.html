使用

webpack方式:

  • npm install burnish-icon --save
  • import 'burnish-icon/style.css';
  • <i class="icon-1" />

普通方式:

  • 下载新版的icon包
  • 引入style.css和font文件夹
  • <i class="icon-1"></i>

制作icon

  • 找设计师做好svg。
  • 到IcoMoon，点击【import icons按钮】 导入项目的selection.json文件。
  • 再导入自己新增的svg文件，点击导出压缩包。
  • fork icon代码仓库。
  • 替换项目中的font文件夹和style.css，测试确认无误后，提pull request。
