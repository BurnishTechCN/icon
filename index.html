使用（WEB端）

webpack方式:

  • npm install burnish-icon --save
  • import 'burnish-icon/css/style.css';
  • import 'burnish-icon/css/ionicons.css';
  • burnish图标：<i class="icon-1" />
  • ionicons图标：<i class="ion-ionic" />

普通方式:

  • 下载新版的icon包
  • 引入style.css、ionicons.css和font文件夹
  • burnish图标：<i class="icon-1" />
  • ionicons图标：<i class="ion-ionic" />

使用（APP 端）

  • 下载新版的icon包
  • 解压获取font文件夹中的字体，如：burnish.ttf，列表可查看图标字符编码如：\e901。
  • 可参考阿里巴巴字体图标加载：http://iconfont.cn/help/iconuse.html

制作icon

  • 找设计师做好svg。
  • Fork icon代码仓库到本地。
  • 在svg目录导入新增的svg文件。
  • 到IcoMoon，点击【import icons按钮】导入所有的svg文件（先删掉所有已有的图标）。
  • 确保导入的图标都处理点亮状态后，点击【generate font】按钮，导出字体图标压缩包。
  • 替换项目中的font、style.css。
  • 修改doc/data.js配置，加入新增的icon class。
  • 执行python -m SimpleHTTPServer 9999，访问http:localhost:9999看效果。
  • 测试无误后，push代码，提交pull request。
