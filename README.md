# icon
font icon 库

# 快速开始

## webpack方式（推荐）

* `npm install burnish-icon`
* `import burnish-styles.css`
* `<i className="icon-1" />`

## 普通方式

* [下载新版的icon包](https://github.com/BurnishTechCN/icon/archive/master.zip)
* 加入font文件夹和style.css文件
* `<link rel="stylesheet" href="burnish-style/style.css" />`
* `<i class="icon-1"></i>`
